## Building the Documentation

The documentation is written in Markdown and can be built into a static website using [MkDocs](https://www.mkdocs.org/):

```bash
# Install MkDocs
pip install mkdocs

# Build the documentation
mkdocs build

# Serve the documentation locally
mkdocs serve
```
